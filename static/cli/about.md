**Loom** is a profile-driven automation runtime that lets you use Lua to define *cli* command trees that perform structured, template-based code generation, embedding, or introspection.

Visit *(https://github.com/taleroangel/loom)* for documentation on how to create your first **profile**

### Need help with the API?
Setup your *Lua* environment by creating a **definition file** using the following command `loom --meta > .loom.d.lua`, this file will be used by your LSP to help you with diagnostics and completion
